## Option 1: Visit the web application at notification-system-vn-ojha.vercel.app

## Or to run it locally follow these steps:-

```bash
# Terminal 1 - Backend
cd backend
npm install
npm start

# Terminal 2 - Frontend (new window)
cd frontend
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.
Click the bell 🔔 and watch it work!

## 📁 What You Have

**Backend (Node.js + Express)**
- Notification API with 5 core endpoints
- Event system (4 scenarios)
- Tenant isolation enforced
- In-memory database with seed data

**Frontend (Next.js + React)**
- Notification bell component
- Unread badge
- Dropdown showing notifications
- Auto-polling every 15 seconds

## 🚀 Quick Test

Once both servers are running:

1. Click the bell 🔔
2. See 3 notifications
3. Click any button ("Member Invited", etc.) on the page
4. Watch new notification appear in the bell immediately
5. Mark some as read
6. Badge updates automatically
7. Try changing tenant/user ID
8. Verify you can only see that tenant's notifications

**That's the whole system working end-to-end!**
